# Книга рецептов (Android, Kotlin, Jetpack Compose)

Цель: приложение для просмотра, поиска и добавления рецептов с локальным хранением, базовой авторизацией, кастомными UI-компонентами, сетевыми данными и обфускацией кода.

## Стек
- Kotlin, Jetpack Compose, Material3
- Navigation Compose
- Room (локальное хранилище)
- DataStore (настройки/токен)
- Retrofit/OkHttp (REST API)
- Coil (изображения)
- R8/ProGuard (обфускация)

## Экраны
1. Home — категории, поиск
2. Catalog — список рецептов
3. Recipe Detail — ингредиенты, шаги
4. Add Recipe — форма добавления
5. Profile/Auth — регистрация/логин

## Кастомные компоненты
- AnimatedFavoriteButton — анимированная кнопка избранного
- ProgressBadge — динамический индикатор сложности/прогресса
- RecipeCard — карточка рецепта с динамическим отображением

## Запуск
1. Откройте проект в Android Studio (Giraffe/новее).
2. Запустите `app` на эмуляторе/устройстве.
3. Release сборка включает обфускацию (R8).

## Структура
- `app/src/main/java/com/example/recipes/...` — код
- `data/local` — Room
- `data/remote` — Retrofit
- `ui/components` — кастомные компоненты
- `ui/screens` — экраны
- `ui/navigation` — навигация

## API
- Пример мок-эндпоинта: https://example.com/api/recipes
- Можно заменить на собственный Node/Mockoon.

## Обфускация
- R8 включен в release.
